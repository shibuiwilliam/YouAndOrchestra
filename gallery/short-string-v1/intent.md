# Intent: Short String v1

A short, elegant Romantic-era string ensemble miniature in A major. Simple, singable melody in Violin I over warm diatonic harmony. Inspired by Tchaikovsky and Dvorak string writing — tender, legato, transparent textures. A single gentle arch from quiet to modest climax and back. ~30 seconds.

## Emotion
- Warm, tender, elegantly romantic
- Gentle nobility without drama

## Purpose
- A self-contained classical miniature
- Demonstrates Romantic-era string writing in its simplest form

## Listening Context
- Concert hall intimacy
- Quiet, contemplative listening

## Key Decisions
- A major chosen for its bright warmth on strings
- 90 BPM allows the melody to breathe without dragging
- 11 bars keeps it concise — a single idea, stated and resolved
- 4-part string writing for classic transparency
- No looping — a miniature with a clear beginning and end
