# YaO Gallery — Pre-generated Examples

These MIDI files are generated by YaO from the included spec templates. Open them in any MIDI player or DAW to hear what YaO produces.

## Examples

| File | Spec | Generator | Notes | Description |
|------|------|-----------|-------|-------------|
| `minimal-piano-rule-based.mid` | `specs/templates/minimal.yaml` | rule_based | 30 | 8-bar solo piano in C major, 120 BPM. Deterministic — always the same output. |
| `minimal-piano-stochastic.mid` | `specs/templates/minimal.yaml` | stochastic (seed=42, temp=0.5) | 31 | Same spec, but with melodic contour, varied rhythms, and velocity humanization. |
| `bgm-90sec-stochastic.mid` | `specs/templates/bgm-90sec.yaml` | stochastic (seed=42, temp=0.5) | 147 | 90-second BGM with piano and acoustic bass across 4 sections. |

## Regenerate

You can regenerate these examples:

```bash
yao compose specs/templates/minimal.yaml
yao compose specs/templates/bgm-90sec.yaml
```

Or create your own:

```bash
yao conduct "a mysterious puzzle game BGM, minimal and looping"
```
