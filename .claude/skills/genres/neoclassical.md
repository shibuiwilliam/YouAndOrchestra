# Neoclassical — Genre Skill

## Tempo
- Range: 60-120 BPM
- Sweet spot: 72-90 BPM (Andante to Moderato)
- Allegro sections (100-120 BPM) for dramatic passages
- Adagio (60-72 BPM) for lyrical, expressive movements
- Tempo changes between sections are idiomatic (rubato, rallentando)

## Key Preferences
- A minor (standard, versatile), D minor (passionate, Beethovenian)
- E minor (melancholic, Chopinesque), C minor (dramatic, stormy)
- Major keys for contrasting middle sections or codas
- Phrygian mode for Spanish-influenced neoclassical passages
- Modulation to relative major or dominant minor between sections

## Iconic Chord Progressions (frequency-ranked)
1. i-iv-V-i (classical minor cadential, ~30%)
2. i-VI-III-VII (Romanticized sequence, ~20%)
3. i-V/iv-iv-V (secondary dominant approach, ~15%)
4. i-ii dim-V-i (diminished passing chord, ~15%)
5. i-bVI-bIII-V (chromatic bass descent, ~10%)
6. i-iv6-V7-i (first inversion subdominant voice leading, ~5%)

## Drum Pattern Family
- Default: null (no drum kit)
- Timpani rolls for dramatic climaxes only
- No percussion in intimate passages
- If rhythmic drive is needed, use pizzicato strings or repeated chords

## Instrumentation Defaults
- Core: violin, viola, cello, piano
- Common additions: contrabass (bass foundation), flute (lyrical countermelody)
- Extended: oboe, clarinet, french_horn (for fuller orchestration)
- Harp for arpeggiated accompaniment figures
- Avoid: all synths, electric instruments, drum kit

## Section Structure
- Exposition: 8-16 bars, present main theme, tonic key
- Development: 16-32 bars, fragment and modulate theme material
- Recapitulation: 8-16 bars, return of theme, possibly varied
- Coda: 4-8 bars, conclusive material
- Alternative: ABA ternary form (contrasting middle section in relative major)

## Trajectory Patterns
- Sonata Arc: tension 0.3 -> 0.7 -> 0.5 -> 0.8 -> 0.3
- Nocturne: tension 0.2 -> 0.4 -> 0.6 -> 0.3 -> 0.2
- Dramatic Climax: tension 0.3 -> 0.5 -> 0.7 -> 0.95 -> 0.2

## Cadences
- Perfect authentic (V-I, root position, soprano on tonic) for structural endings
- Half cadence (phrase ending on V) for mid-section pauses
- Deceptive (V-VI) for extending phrases unexpectedly
- Plagal (IV-I) reserved for final "amen" codas
- Phrygian half cadence (iv6-V in minor) for Spanish color

## Cliches to AVOID
- Power chords or parallel fifths (violates voice-leading rules)
- Drum kit patterns of any kind
- Synth pads substituting for real string writing
- Block chord accompaniment without any voice movement
- Ignoring counterpoint (at least 2 independent voices expected)
- Excessive repetition without development (classical expectation is variation)

## Quality Heuristics
- Voice leading: stepwise motion preferred, leaps resolved by step
- At least two independent contrapuntal voices at all times
- Dynamics: wide range from pp to ff with gradual transitions
- Register: use full range of each instrument (avoid mid-range clustering)
- Articulation variety: legato, staccato, marcato should all appear
- Motivic development: themes should be fragmented, inverted, or augmented
- Avoid parallel octaves and fifths between any two voices

## Use Cases
- Classical crossover albums
- Film/game scores with period settings
- Solo piano or chamber ensemble pieces
- Concert recital programming
