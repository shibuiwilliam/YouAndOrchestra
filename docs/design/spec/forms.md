# Design: Form Library + Diversity Sources (Phase γ.6)

## Problem
Mode collapse: every piece sounds structurally similar regardless of intent.

## Solution
- Form Library: 20 SongForm definitions (AABA, rondo, through-composed, blues, J-pop, etc.)
- 8 Melodic Generation Strategies: contour_based, motif_development, linear_voice, arpeggiated, scalar_runs, call_response, pedal_tone, hocketing
- Each strategy produces distinct character, verified by tests
